
# Simple Text-to-Speech (Python)

This is a small learning project where I explored how text-to-speech works in Python using `pyttsx3`.

## What it does

* Takes user input
* Converts text into speech
* Runs in a loop until stopped

## Why I built this

I wanted to understand how systems convert text into audio and how basic voice assistants work at a low level.

## Key learning

* Handling repeated execution in loops
* Managing speech engine state in Windows
* Making input case-insensitive for better UX

## Future ideas

* Add voice input (speech recognition)
* Turn this into a basic assistant
* Integrate with APIs for real-world actions
